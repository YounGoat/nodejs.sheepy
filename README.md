#	sheepy

##	Install & Run

```bash
# Install
npm install -g sheepy

# Change to root directory
cd /path/to/root/dir/

# Create HTTP server on port 8080
sheepy 8080
```

##	Other Ways

There are many other ways to serve static files via HTTP.

*	__Python SimpleHTTPServer__

	```bash
	python -m SimpleHTTPServer 8080
	```
